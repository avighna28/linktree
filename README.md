# 🎓 Placement Roadmap Hub

A premium, all-in-one resource library managed by **Aarti Jha** and **Avighna** to help students navigate the stress of placement preparation and academic planning.

---

## 🌟 The Story Behind the Project

This project wasn't born out of a simple assignment—it was born out of a real problem we both faced. 

During our own preparation, we realized how difficult and distracting it is to search for quality roadmaps, study guides, and coding sheets across the vast internet. The constant switching between tabs and the struggle to find "the right" resource often led to more confusion than learning.

We decided to build a platform that acts as a **Single Source of Truth** for every student. Our goal is to ensure that no one has to waste time searching or get distracted while preparing for their future.

### 🤝 A True Partnership
This project is a result of a deep collaboration where every decision was made together. 

*   **The Spark**: The core idea was envisioned by **Aarti Jha**. She recognized the gap in how students access resources and brought the vision to the table.
*   **The Evolution**: **Avighna** took that spark, added key architectural changes, and refined the idea into a high-performance web application.
*   **The Execution**: Both worked side-by-side to implement the design, gather the resources, and build the premium "Placement Roadmap Hub" you see today.

There isn't just one person behind this; it is a shared mission to make student life a little bit easier.

---

## 🚀 Key Features

*   **35+ Curated Resources**: Hand-picked roadmaps, DSA sheets, and interview guides.
*   **Local Storage**: All resources are hosted locally within the project to ensure 100% uptime and fast loading.
*   **Premium Glassmorphism UI**: A modern, dark-themed interface built for focus and aesthetics.
*   **In-Page Previews**: View resources directly without leaving the hub.
*   **Full Search Functionality**: Instant filtering to find exactly what you need.
*   **Responsive Power**: Optimized for everything from high-end desktops to mobile screens.
